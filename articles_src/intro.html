Ceci est mon petit blog comme exercice de php. Chaque article est un petit script en php qui

  • Lit le fichier rss.xml,
  • Crée la barre sur le côté avec la liste des articles,
  • ajoute la date de publication et le titre,
  • inclus le code html de l'article proprement dit.
En réalité les informations sont données à un script en python qui génère le code php qui fait tout cela.

Les sources sont disponibles sur github.
